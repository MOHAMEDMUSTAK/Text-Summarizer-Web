# AI Text Summarizer (Web App)
## Overview
A web-based AI text summarization tool built using Python and Streamlit.
The application performs extractive summarization using frequency-based scoring.
## Features
- Interactive web interface
- Adjustable summary length
- Extractive NLP-based summarization
- Clean and simple UI
## Tech Stack
- Python
- Streamlit
- Regular Expressions
- Collections (Counter)
## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the application:
   streamlit run app.py
3. Open the local browser link provided.
